ReelMind

Personal ProjectJun 2026

An end-to-end RAG pipeline over 1,000+ IMDb movies that semantically retrieves and answers queries using Sentence Transformers embeddings, a Pinecone vector database, and Llama 3.3 70B via Groq API.

Overview

ReelMind is a semantic search engine over a database of 1,000+ movies. Instead of keyword matching, it embeds each movie's metadata and plot summary into vector space using Sentence Transformers, stores those vectors in Pinecone, and uses Llama 3.3 70B (via the Groq API) to generate natural-language answers grounded in the retrieved results.

The app is deployed live on Streamlit Cloud, with persistent vector storage so the index survives redeploys and cold starts.

Engineering Decisions

  • Migrated from ChromaDB to Pinecone. Streamlit Cloud's ephemeral filesystem wiped the local vector index on every redeploy, so a managed, persistent store was needed to survive that.
  • Resolved cold-start failures. Restructured how the embedding model and vector client were initialized at app startup, eliminating timeout errors on the first request after a deploy.
  • Chose Groq for inference. Serves Llama 3.3 70B responses with low latency, keeping the semantic search feel responsive despite a large underlying model.
