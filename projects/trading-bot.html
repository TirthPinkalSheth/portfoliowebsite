AI Trading Bot via MCP Server

Personal ProjectMar 2026

A Model Context Protocol (MCP) server built in TypeScript that lets Claude AI autonomously place live LIMIT orders on the NSE via the Zerodha KiteConnect API.

Overview

This project exposes a set of buy/sell tools over the Model Context Protocol, so Claude Desktop can interpret a natural-language trading instruction and execute it as a real limit order on the National Stock Exchange, through Zerodha's KiteConnect API.

The server implements Zerodha's session-based OAuth login flow to establish a trading session. KiteConnect access tokens expire daily with no refresh-token support, so the token is handled manually each session, and the whole flow was validated with real money on the live market.

Engineering Decisions

  • Built on the Anthropic MCP SDK. Exposes trading actions as tools Claude could call directly, rather than building a custom chat wrapper around the broker API.
  • Used Zod for schema validation. Every tool input is checked so malformed or ambiguous natural-language-derived orders are rejected before they ever reach the broker API.
  • Implemented Zerodha's session-based OAuth flow. KiteConnect access tokens expire daily with no refresh-token support, so the login flow is completed manually each trading session to generate a new token.
